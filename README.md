# halviewer

graphical halviewer for linuxcnc

![screenshot](./halviewer.png)

## install depends
```
apt-get install python3-pyqt5 python3-graphviz
```

## before running halviewer, you need to run linuxcnc, than:
```
python3 halviewer.py
```
